Sequential Search

We will introduce you to a method of searching known as sequential search

Looping

Here we will need to know a bit about looping, the main page on looping can be found here

fruits = ['apples', 'oranges', 'pears', 'bananas', 'apricots', 'melons']

fruits.each do |fruit|

if fruit == 'pears'

print "there are pears in the list"

end

This will loop through the list and if the desired item is present then the program will print to the console confirming the item we are looking is there
