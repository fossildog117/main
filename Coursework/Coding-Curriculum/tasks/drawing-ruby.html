Drawing A Ruby

It is very useful to be able to draw images with code, it allows you to create user interfaces so people can use your programs through a nice design as opposed to just using commands

As you can see above, co-ordinates for most graphics library are different to what you learn at school with (0,0) being the top left corner rather than bottom left. You could consider it as being the bottom right quadrant of a normal graph but with y increasing as you go down rather than decreasing


There are only a couple of functions defined for drawing in this challenge

drawLine(x1,y1,x2,y2)

This will draw a line from (x1, y1) to (x2, y2)

fillRect(x,y,width,height)

This will draw a filled rectangle with top left at (x,y) with a defined width and height

clearCanvas

This will reset the canvas

These functions, unlike the functions that we have used so far, take parameters which are listed in the same order in brackets after the name so for example:

fillRect(0,0,200,200)

Will draw a filled rectange with top left at (0,0), width 200 and height 200.

Whilst there are only a few functions defined here they can be very powerful. However to create elaborate images it can be quite long and tiresome but the next task will show you how to create your own functions to draw shapes but for now can you draw a ruby using just these functions? The canvas is 400 by 400.
