If statements

We will introduce you to conditionals. These help the computer make informed decisions and we will start off with some of the basics of conditionals

The most basic conditional statement is called an 'if statement', they are defined like so

x = 1

if x > 1

print "x is greater than 1"

elsif x < -1 and x != 0

print "x is a negative number smaller than -1"

else

print "x is 1"

end

This will print "x is 1". The program will first evaluate if x is greater than 1, then evaluate if x is 0 or smaller than 1, if both are false then it evaluate x to 1 since the conditionals have filtered out the possibility of x being greater than 1 and smaller than 1 therefore the only number x can be is 1.
