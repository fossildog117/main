Calculator

For this task you will need your own Ruby compiler and text-editor.

Functions

In order to create a simple calculator, we can use def and end keywords to write several functions of your own. For more details about this refer to the Defining Shapes task.

def hello

'Hello and Welcome'

end


puts hello

Example: This puts statement outputs the result of the hello function: 'Hello and Welcome'.


We can also declare parameters (a list of local variable names) of the function in parentheses after the name of the function in the def statement.

def helloName (name)

'Hello ' + name

end


puts(helloName('Peter'))

Example: The helloName function in this instance takes a parameter called Peter and prints Hello Peter. Noticed we have used the ‘+’ operator to carry out string concatenation (combine the strings Hello and the name parameter).


String Interpolation

String interpolation allows us to replace placeholders in a string with values. Any code placed inside #{ } will be evaluated even if the placeholders aren’t variables. We use double quotes for string interpolation as single quotes do not support it. This is because double quotes allow escape sequences to be used while single quotes do not. In double quotes, “\n” is interpreted as a new line and appears as a new line when run, whereas '\n' displays the actual escape sequence character to the user.

a = 10

b = 15

puts "The number #{a} is less than #{b}"

Example: It outputs 'The number 10 is less than 15'.


Getting Input

The statement gets is what lets the program read input from the keyboard. The function gets returns a string and a '\n' character (when you press Enter). This means that when you print the value retrieved by gets in a string, it will include the new line character. This makes your string go on to new line. In order to avoid this, use the string function chomp to remove this new line character.

gets.chomp

Use this to retrieve inputted strings

In order to retrieve intergers inputted, you need to convert the string to an integer by using the to_i function.

gets.to_i

Use this to retrieve number (integer) inputs
