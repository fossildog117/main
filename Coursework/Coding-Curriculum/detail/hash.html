Hashes

In Ruby, a hash is another form of data structure. You access values from a hash by using a key which can be any string or number. The syntax used for hashes is show below. We use curly brackets to enclose the hash.

d= {'key1' => 1 , 'key2' => 2 , 'key3' => 3}

Example: Hash syntax.

The hash above is named 'd' and it has 3 'key and value' pairs. The key 'key1' points to the value 1, 'key2' to 2, and so on. We place a '=>' in between the key and value and ensure that each pair of key and value is separated by commas. The key is enclosed in quotes to represent it is a string. An empty pair of curly braces {} is an empty hash.

Hashes are ideal for phone books because you can pair a name with a phone number and for login pages by pairing an e-mail address with a username.


New Hash

You can create a new hash by using the 'Hash.new' method. We can assign a variable to Hash.new and this creates a new empty hash. It has the same effect as setting the variable to equal empty curly braces. An example is shown below:

space=Hash.new

space["key"]="value"

Example: Creating and assigning to a hash.


Printing a Hash

Below is an example showing how you can print out values from a hash. You state the name of the hash (spaceAgency) and then write within the square brackets the key of the value that you want to print out.

spaceAgency = {'USA' => 'NASA', 'Europe' => 'ESA', 'India' => 'ISRA'}

puts "#{spaceAgency['USA']}"

puts "#{spaceAgency['Europe']}"

Example: Printing from a hash. Above prints NASA and ESA.


Adding and Removing Items

You can add values to a hash once created by using the syntax below:

hash_name[new_key]=new_value

Adding values to an exiting hash

A new value can be associated with a key that already exists by assigning a value to the key, like below:

hash_name[key]=new_value

Reassigning values to a key.

Items can be removed from a hash by using the .delete method as shown below. This command will remove the key 'key_name' as well as its associated value from the hash.

hash_name.delete(key_name)

Length function and Arrays

We can identify the number of key-value pairs in a hash by using the 'length' function. It is possible to have arrays within a hash as shown in the example below. Each pair counts as one, even if the value is an array. For more notes on arrays, read up earlier notes.

spaceCraft = {}

spaceCraft [1969] ='Apollo 11'

spaceCraft[1989]='Galileo'

spaceCraft[1977]='Voyager 1'

puts "There are #{spaceCraft.length} items in the spacecraft hash."

puts spaceCraft.inspect

Example: We first create an empty hash and add new key-value pairs. Then we .lenght method and print hash.

We used the .inspect method to print the spaceCraft hash. It converts the hash into a string and prints it in the form a hash.

The output of the above code is:

There are 3 items in the spacecraft hash.

{1969=>"Apollo 11", 1989=>"Galileo", 1977=>"Voyager 1"}

Output of above example

For more details on loops, read up earlier notes on loops.
