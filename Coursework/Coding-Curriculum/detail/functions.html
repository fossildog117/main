Functions

Functions allow a section of code to be encapsulated so that it can be used multiple times to avoid duplication of code and make the code easier to read

To define a function 'def' is used followed by the function name and then the code that follows is the function until an 'end' statement that is not closing a block within the function.

def function

// function code

end

This defines a function called function.

A function can take parameters, which allows it to be given values when it is used, by including names for the parameters in brackets after its name separated by commas.

def add(x, y)

x + y

end

This defines a function add which takes two variables and return the result of adding them

Functions either return a value or cause side-effects. So for example you could have a function that adds two numbers and then you could make that return the result of the addition, this can be done explicitly using a 'return' statement or otherwise the last statement in the code will be evaluated as what will be returned. A function otherwise could cause side-effects such as a drawLine function could draw a line in a graphical environment, but would have no meaningful return value

def subtract(a, b)

a - b

end


def multiply(a, b)

z = a * b

return z

end

This defines a function subtract that takes two parameters and returns the first minus the second, and a function multiply which takes two parameters and returns their product

Normally when using functions the code of the function is indented as shown in these examples so that it is easier to read. As well as this if a function is returning a boolean value its name is normally ended with a '?' to indicate this to make it easier to make sure the functions are being used correctly by indicating return type in the name.

def isPositive?(x)

x >= 0

end

This defines a function isPositive? which takes a value and returns true if it greater than or equal to zero and false otherwise

Functions are called by just typing their name, and if they take parameters they are included in brackets separated by commas.

a = add(3, 5)

b = subtract(a, 5)

Here a variable a is declared that is the result of calling the add function on 3 and 5 which is 8, and then a variable b is declared that is the result of calling subtract on a and 5 which is 3
